# le-voyage-a-stockholm
A small app to present my favorite places in Stockholm on a map
## Built with 

### - client-side -
- React
- CSS & SCSS (SASS)
- [@react-google-maps/api](https://www.npmjs.com/package/@react-google-maps/api)
  
### - server-side -
